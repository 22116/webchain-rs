Emerald offline wallet command line interface.

Usage:
  emerald server [--port <port>] [--host <host>]
  emerald account list [--chain <name>] [--chain-id <id>]
  emerald account new [--chain <name>] [--chain-id <id>] [<name> [<description>]]
  emerald account update [--chain <name>] [--chain-id <id>] (<uuid>|<address>) [<name> [<description>]]
  emerald account shake [--chain <name>] [--chain-id <id>] (<uuid>|<address>)
  emerald account hide [--chain <name>] [--chain-id <id>] <uuid>|<address>
  emerald account reveal [--chain <name>] [--chain-id <id>] <uuid>|<address>
  emerald account import [--chain <name>] [--chain-id <id>] [<path>...]
  emerald account export [--chain <name>] [--chain-id <id>] <uuid>|<address>
  emerald transaction sign [--chain <name>] [--chain-id <id>] <from> <nonce> <to> <gas> <gas-price> (<value>|<data>)
  emerald -V | --version
  emerald -h | --help

Options:
  -h, --help                                Show this message
  -V, --version                             Show current version
  -v, --verbose...                          Use verbose output (-vv very verbose output)
  -q, --quiet                               No output printed to stdout
      --host <host>                         Listen host [default: 127.0.0.1]
      --port <port>                         Listen port [default: 1920]
      --base-path <path>                    Base directory path, if omitted default os-specific value will be used:
                                              + Mac OS X: ~/Library/Emerald
                                              + Linux: ~/.emerald
                                              + Windows: %USERDIR%\.emerald
      --security-level normal|high|ultra    Level of security for cryptographic operations [default: normal]
  -c, --chain mainnet|testnet               Chain name [default: mainnet]
      --chain-id <id>                       Chain id [default: 61]
