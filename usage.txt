Emerald offline wallet command line interface.

Usage:
  emerald server [OPTION] [--port <port>] [--host <host>]
  emerald account list [OPTION] [--show-hidden]
  emerald account new [OPTION] [--security-level normal|high|ultra] [<name> [<description>]] [FILE]
  emerald account shake [OPTION] [--security-level normal|high|ultra] (<uuid>|<address>) [FILE...]
  emerald account update [OPTION] (<uuid>|<address>) [<name> [<description>]]
  emerald account hide [OPTION] (<uuid> | <address>)...
  emerald account reveal [OPTION] (<uuid> | <address>)...
  emerald account import [OPTION] [FILE...]
  emerald account export [OPTION] (<uuid> | <address>)... [FILE...]
  emerald transaction sign [OPTION] <from> <nonce> <to> <gas> <gas-price> (<value>|<data>)
  emerald -V | --version
  emerald -h | --help

Options:
  -h, --help                                Show this message
  -V, --version                             Show current version
  -v, --verbose...                          Use verbose output (-vv very verbose output)
  -q, --quiet                               No output printed to stdout
      --host <host>                         Listen host [default: 127.0.0.1]
      --port <port>                         Listen port [default: 1920]
      --base-path <path>                    Base directory path, if omitted default os-specific value will be used:
                                              + Mac OS X: ~/Library/Emerald
                                              + Linux: ~/.emerald
                                              + Windows: %USERDIR%\.emerald
      --show-hidden                         Show also hidden accounts
      --security-level normal|high|ultra    Level of security for cryptographic operations [default: normal]
  -c, --chain mainnet|testnet               Chain name [default: mainnet]
      --chain-id <id>                       Chain id [default: 61]
