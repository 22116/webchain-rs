Emerald offline wallet command line interface.

Usage:
  emerald server [options] [--port <port>] [--host <host>]
  emerald account list [options]
  emerald account new [options] [--security-level normal|high|ultra] [<name> [<description>]]
  emerald account shake [options] [--security-level normal|high|ultra] (<uuid>|<address>)
  emerald account update [options] (<uuid>|<address>) [<name> [<description>]]
  emerald account hide [options] <uuid>|<address>
  emerald account reveal [options] <uuid>|<address>
  emerald account import [options] [<path>...]
  emerald account export [options] <uuid>|<address>
  emerald transaction sign [options] <from> <nonce> <to> <gas> <gas-price> (<value>|<data>)
  emerald -V | --version
  emerald -h | --help

Options:
  -h, --help                                Show this message
  -V, --version                             Show current version
  -v, --verbose...                          Use verbose output (-vv very verbose output)
  -q, --quiet                               No output printed to stdout
      --host <host>                         Listen host [default: 127.0.0.1]
      --port <port>                         Listen port [default: 1920]
      --base-path <path>                    Base directory path, if omitted default os-specific value will be used:
                                              + Mac OS X: ~/Library/Emerald
                                              + Linux: ~/.emerald
                                              + Windows: %USERDIR%\.emerald
      --security-level normal|high|ultra    Level of security for cryptographic operations [default: normal]
  -c, --chain mainnet|testnet               Chain name [default: mainnet]
      --chain-id <id>                       Chain id [default: 61]
